📦 Container Escalada

Participar do pedido container Escalada
Contatar o time Escalada para entrar no próximo pedido de container coletivo. Entender processo: prazos, formas de pagamento, logística de recebimento e armazenagem.
⏱️ 3-4 horas💰 R$ 8.000 (capital container)🔥 Principal ação!
Selecionar 5-8 produtos para importar
Com base nos dados de 5 meses, selecionar produtos com: alta demanda comprovada, boa margem, fornecedor confiável na China, sem problemas com Express Remessa anterior.
⏱️ 3-4 horas💰 R$ 0 (análise)
Calcular volumes e custos
Para cada produto: calcular MOQ no container, custo CIF, impostos (60% + ICMS), frete doméstico, custo final unitário. Confirmar margem mínima de 40% após todos os custos.
⏱️ 4-5 horas💰 R$ 0
Validar margem pós-importação
Simular cenários pessimista/realista/otimista para cada produto do container. Garantir que mesmo no cenário pessimista a operação é rentável. Aprovação final.
⏱️ 2-3 horas💰 R$ 0
Preparar capital para o container
Reservar R$ 8.000 do faturamento para o container. Verificar timing de pagamento vs recebimento de vendas. Não comprometer capital de giro operacional.
⏱️ 1-2 horas💰 R$ 8.000
